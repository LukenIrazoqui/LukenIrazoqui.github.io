EasySave

This desktop application was done by a team of 5 and allows users to :

  • Create
  • Delete
  • Launch
  • Pause
  • Resume
  • Stop

backup jobs.

This program uses threads, which allows the launch or multiple back-up jobs simultaneously.

It also allows users to encrypt files with the chosen extensions.

Finally, a remote EasySave application permits users to control the main application from another location.

Used technologies :

  • C#
  • .NET
  • GitHub
  • Visual Studio
Please refer to the GitHub for more information.
