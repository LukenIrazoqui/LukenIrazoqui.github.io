VRP problem solution

This desktop application was done by a team of 5 to find a solution to the famous VRP problem

After a mathematical analysis of the problem, the group went on to create 2 algorithms.

First, we thought about dividing the plane area by the number of trucks and using a greedy algorithm to solve the TSP problem for each group.

Then, we used a genetic algorithm to find the best solution by applying mutations and crossovers.

Used technologies :

  • Python
  • Jupyter Notebook
